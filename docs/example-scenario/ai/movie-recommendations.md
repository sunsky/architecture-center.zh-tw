<span data-ttu-id="777dc-117">此案例在電影評分資料集上使用 Spark [替代最小平方][als] (ALS) 演算法，來訓練和評估機器學習模型。</span><span class="sxs-lookup"><span data-stu-id="777dc-117">This scenario covers the training and evaluating of the machine learning model using the Spark [alternating least squares][als] (ALS) algorithm on a dataset of movie ratings.</span></span> <span data-ttu-id="777dc-118">此案例的步驟如下：</span><span class="sxs-lookup"><span data-stu-id="777dc-118">The steps for this scenario are as following:</span></span>

1. <span data-ttu-id="777dc-119">前端網站或應用程式服務會收集使用者與影片互動的歷程記錄資料，這些資料會顯示於使用者、項目和數字評分元組的資料表中。</span><span class="sxs-lookup"><span data-stu-id="777dc-119">The front-end website or app service collects historical data of user-movie interactions, which are represented in a table of user, item, and numerical rating tuples.</span></span>
